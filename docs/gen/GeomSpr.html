Class: GeomSpr

GeomSpr

A Phaser Sprite extension to make primitive shapes as sprites.

new GeomSpr(game, x, y, type, path [, opt])

Create geometric sprite shapes using Phaser graphics.

Parameters:
Name Type Argument Description
game Phaser.Game

A Phaser Game instance.
x Number

x coordinate.
y Number

y coordinate.
type string

Type of Shape ('circle', 'rectangle' or 'polygon').
path array

Data to draw shapes. For polygon accepts any polygon construction setup.*
opt Object <optional>

An optional list of options.

Properties
Name Type Argument Default Description
bgColor Hex <optional>
 0xFFFFFF

Shape's background color.
aX Number <optional>
 0.5

Anchor in x axis.
aY Number <optional>
 0.5

Anchor in y axis.
alpha Number <optional>
 1

Alpha value.
outLineData Object <optional>

Arguments to build outline border.

Properties
Name Type Description
width Number

Outline width.
color Hex

Outline color.

Assign a parent if you setted it in this property. *Path will accept any, an array or the Phaser shapes itself. In case of polygon, it also accept all formats available to construct Phaser.Polygon
